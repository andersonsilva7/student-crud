Página inicial do sistema.

Descrição do Sistema

Esse projeto tem o objetivo de manipular uma lista de estudantes.
É um simples sistema CRUD (CREATE, READ, UPDATE and DELETE), utilizando um JSON Server para armazenar os dados. Aqui, conseguimos realizar as seguintes operações:

- Adicionar um aluno.
- Exibir um aluno.
- Editar um aluno.
- Excluir um aluno.
